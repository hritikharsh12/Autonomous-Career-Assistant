# 🤖 Autonomous Career Assistant

An AI-powered career assistant that analyzes resumes, matches jobs, and improves ATS performance using **LangGraph, HuggingFace, ChromaDB, FastAPI, and Django**.

---

## 🚀 Key Highlights

- 🎯 Matches profiles to jobs with **~85% semantic similarity**
- ⏱️ Reduces job search time by **~60%**
- 📄 Improves ATS keyword density by **~40%**
- ⚡ Scrapes **50+ live job listings in under 30 seconds**
- 🔁 Automated feedback loop for resume improvement

---

## 🧠 Core Features

- Resume parsing (PDF, DOCX, TXT)
- RAG-based job matching using ChromaDB
- ATS scoring & keyword gap detection
- AI-powered resume rewriting (LLM)
- Async job scraping from multiple platforms
- End-to-end pipeline using LangGraph
- Full-stack integration (FastAPI + Django)
